[alpine.js](https://alpinejs.dev) 찍먹

* x-data : 컨테이너에 삽입되는 렌더링용 데이터
* x-ref : `$ref.변수`. 단일 변수만, 배열형태 안됨
* x-text : `$ref.변수`의 내용을 `innerText`에 삽입

* @click : 클릭 이벤트

* $refs: `x-ref`로 선언되는 변수들
* $el: 이벤트 발생시의 해당 엘리먼트 (v3)
* $root: `x-data` 컨테이너
