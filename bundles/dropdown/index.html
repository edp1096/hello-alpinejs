Shadow DOM을 적용한 드롭다운 컴포넌트

이 예제는 여러 개의 드롭다운 컴포넌트가 있어도 아이콘 라이브러리를 한 번만 로드합니다.

Shadow DOM의 장점:
  • 완전한 스타일 격리: 외부 CSS의 영향을 받지 않음
  • DOM 캡슐화: 컴포넌트 내부 구조가 외부와 분리됨
  • 이름 충돌 방지: ID나 클래스 이름이 겹칠 걱정 없음
  • 중복 로딩 방지: 아이콘 라이브러리를 공유하여 자원 절약

여러 드롭다운 컴포넌트를 함께 사용

개발자 도구 → 네트워크 탭을 확인해보세요. RemixIcon은 한 번만 로드됩니다.
